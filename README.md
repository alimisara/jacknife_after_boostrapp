# Code du projet Jackniffe_after_boostrapp: Statistique Computationnelle

Ce projet est l'application de la méthode Jackniffe_after_boostrapp dans le cadre de l'analyse et présentation de l'article d'Effron

La méthode est appliquée sur des données simulées suivant la loi normale, en utilisant la variance comme la statistique d'intérêt.
